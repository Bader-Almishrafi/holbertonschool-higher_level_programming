The ``add_integer`` function adds two numbers (integers or floats).

>>> add_integer = __import__('0-add_integer').add_integer

Regular addition
>>> add_integer(1, 2)
3
>>> add_integer(100, -2)
98
>>> add_integer(2)

Integer casting (floats are cast to int)
>>> add_integer(1.4, 2)
3
>>> add_integer(100.3, -2)
98
